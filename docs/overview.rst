DMI Plugin Overview
===================

.. warning:: draft

The Data-Model-Inventory (DMI) Plugin is a rest interface used to synchronize CM-Handles data between CPS and DMI through the DMI Plugin.
This is built previously from the CPS-NF-Proxy component.
