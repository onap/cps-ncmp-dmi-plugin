Introduction
============

Several DMI APIs use the CM Handle parameter.
The CM Handle parameter is used for querying specific CM Handle ID's. CM-Handle is inspired by the `XML Path Language (XPath) 3.1. <https://www.w3.org/TR/2017/REC-xpath-31-20170321/>`_

This section describes the functionality currently supported by CPS Path.
